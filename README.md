# ykb-order-ios-case
Simple eCommerce App (Swift, UIKit, MVVM, Combine, Protocol Oriented, No Storyboard)

## Options:
- Login
- Product List
- Product Detail
- Add to Cart
- Stepper Max Value Set to Quantity of Product
- Order

## Used Libraries:
- Alamofire
- SnapKit
- Kingfisher
- Fake Api: https://dummyjson.com
