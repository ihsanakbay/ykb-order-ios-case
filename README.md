# ykb-order-ios-case
Simple eCommerce App (Swift, UIKit, MVVM, Combine, Protocol Oriented, No Storyboard)

## Options:
- Login
- Product List
- Product Detail
- Add to Cart
- User cannot add more products than stock amount
- Order

## Used Libraries:
- Alamofire
- SnapKit
- Kingfisher
- Fake Api: https://dummyjson.com
